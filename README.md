# Student Management System

## Overview
The Student Management System is a console-based application developed using Python. 
It helps manage student records efficiently by allowing users to add, update, delete, and search student information.

This project demonstrates the practical use of Python programming concepts and file handling.

## Features
- Add new student records
- Update existing student information
- Delete student data
- Search student records
- Store data using file handling
- Simple and user-friendly interface

## Technologies Used
- Python
- File Handling
- Object-Oriented Programming (OOP)

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/msingh-11/student-management-system
2. Open the project folder
3. Run the file using:
   python main.py
